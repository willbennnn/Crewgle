# Stack
List of frameworks, APIs, Languages, etc. used for building the project, and why they're being used.

## Backend
- C++
- Drogon

Allows for the buiding of a good backend in C++ without having to start completely from scratch. Good for learning the basics.  

## Database
- SQLite

Very simple setup, good for local development of a student built project (maybe update to PostgreSQL later.  

## Frontend
- React
- Vite
- TypeScript

Common modern frontend stack (thus lots of support online), lots of learning resources, quick development.

## Version Control / Collaboration
- GitHub

Easy collaboration with branches, pull requests, and issues.  

## Future Tools
- Docker  

Making setup uniform for everyone later on
